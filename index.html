CSS Frameworks

What is a CSS Framework?
A collection of files that makes it far easier to style your website
Mention that it makes it easier to style the website, and most of them also provide grid layouts and assistance with mobile websites
Strapless
Bootstrap
Materialize CSS (The Best One)

Add the necessary Scripts / Stylesheets


                                    ...
                                    <head>
                                        <!-- Let browser know website is optimized for mobile -->
                                        <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
                                        <!-- Import Google Icon Font -->
                                        <link href="https://fonts.googleapis.com/icon?family=Material+Icons"
                                              rel="stylesheet" />
                                        <!-- Import materialize.css -->
                                        <link type="text/css"
                                              rel="stylesheet"
                                              href="https://cdnjs.cloudflare.com/ajax/libs/materialize/0.100.2/css/materialize.min.css"
                                              media="screen,projection"/>
                                        <title>My Fourth Webpage</title>
                                    </head>

                                    <body>
                                        ...
                                        <!-- Import jQuery -->
                                        <script type="text/javascript" src="https://code.jquery.com/jquery-3.2.1.min.js"></script>
                                        <!-- Import materialize.js -->
                                        <script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs/materialize/0.100.2/js/materialize.min.js"></script>
                                    </body>
                                    ...
